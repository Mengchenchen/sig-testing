### 添加节点

1. 添加节点-1

|||
| ---- | ---- |
| 编号 | 集群管理-集群-添加节点-1 |
| 测试点 | 1. 添加节点 |
| 预置条件 | 1. 集群列表存在集群demo\
2. 节点列表存在一个可用节点caasnode-1 |
| 步骤 | 1. 在集群管理-集群页面，点击集群demo\
2. 点击“节点列表”\
3. 点击“添加节点”\
4. 勾选caasnode-1到工作节点，点击“确定”按钮\
5. 查看节点列表 |
| 预期结果 | 1. 步骤1中，跳转到集群详情页面\
2. 步骤2中，跳转到节点列表页\
3. 步骤3中，跳转到添加节点页\
4. 步骤4中，添加节点成功\
5. 步骤5中，节点列表新增节点caasnode-1,集群状态从“更新中”改为“运行中” |
