#### 角色详情

1. 编辑角色-1

|||
| ---- | ---- |
| 编号 | 访问控制-角色-角色详情-编辑角色-1 |
| 测试点 | 1. 编辑角色 |
| 预置条件 | 1. 角色列表存在创建的角色'test-role'，无描述 |
| 步骤 | 1. 在访问控制-角色页面，点击角色test-role\
2. 点击“编辑”按钮\
3. 输入描述'test-des'，点击“确定”按钮\
4. 查看角色列表 |
| 预期结果 | 1. 步骤1中，跳转到角色详情页\
2. 步骤2中，弹出编辑角色弹窗\
3. 步骤3中，编辑角色成功\
4. 步骤4中，角色test-role描述更新为test-des |
